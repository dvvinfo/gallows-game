# gallows-game

Игра "Виселица", реализованная на Vue 3 и TypeScript.


[Поиграть ](https://gallows-game.netlify.app/) в игру  "Виселицу".


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
